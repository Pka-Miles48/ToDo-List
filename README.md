# CPW 204 ToDo List
A straightforward To-Do website.

## Live website
Check out the live website, which is hosted by [live website](https://pka-miles48.github.io/CPW204---ToDo-List/)

## Features
- Checkboxes for the dates
- A marker (for checkmarking the boxes)
- A date
- An Add Text button (for the tasks)
- An eraser
- A print button
- A highlighter
- A translator
